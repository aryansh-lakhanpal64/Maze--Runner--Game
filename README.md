# Maze Runner

Maze Runner is a GUI-based Java application that visualizes pathfinding algorithms in a maze environment. It demonstrates algorithms like BFS, DFS, and Dijkstra using an interactive grid.

## Features
- BFS, DFS, and Dijkstra algorithms
- Interactive grid (start, end, walls)
- Color-based path visualization
- Step-by-step animation
- Execution time and performance tracking
- Simple and user-friendly GUI (Java Swing)

## Technologies Used
- Java
- Java Swing
- Data Structures (Queue, Stack, Priority Queue)

## How to Run
1. Open the project in any Java IDE (IntelliJ / Eclipse / VS Code)

2. Compile and run:
   javac Main.java
   java Main

## Usage
- Select start and end points
- Add walls in the grid
- Choose algorithm (BFS / DFS / Dijkstra)
- Run to visualize the path
