# Maze Runner: A Level-Based Pathfinding Game

A Java Swing GUI-based maze pathfinding visualizer with animated algorithm execution.

## Features
- BFS, DFS, and Dijkstra algorithms
- Start and End point selection
- Wall and Mud editing
- Random maze generation
- Easy / Medium / Hard with 5 sublevels each
- Animated visited cells and final path
- Speed control slider
- Execution time and frontier statistics

## How to Run
Compile all files:

```bash
javac *.java
```

Run the program:

```bash
java Main
```
